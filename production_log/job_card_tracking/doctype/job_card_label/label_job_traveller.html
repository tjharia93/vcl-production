SECTION 6 · OPERATOR SOP (LABEL PRESS)
A. PRE-RUN SETUP
  1. Verify job card details against approved customer artwork & this traveller.
  2. Confirm substrate reel matches {{ doc.material_type or "spec" }}, width and batch.
  3. Mount plates by code; check for damage, edge chips, or residual ink.
  4. For UV jobs ({{ "YES" if doc.ink_type == "Process UV" else "N/A" }}): verify lamp hours < 1200h; confirm dichroic reflectors clean.
  5. Set anilox roll by BCM / screen count per ink deck; record in log.
  6. Mix inks to recipe shown in Print Colours above; verify viscosity (Zahn #2) and tack.
  7. Load die-cut tooling; confirm cylinder teeth = {{ doc.cylinder_teeth or "—" }}.
B. FIRST ARTICLE
  1. Produce a short proof run (≥ 50 labels) at reduced speed.
  2. Check registration on all {{ doc.number_of_colours or "—" }} colour{% if (doc.number_of_colours or 0) != 1 %}s{% endif %}; tolerance ≤ 0.15 mm.
  3. Densitometer read: ΔE ≤ 2.0 vs. approved swatch for each spot colour.
  4. Barcode verification: ANSI grade B or better on every SKU.
  5. Die-cut test: clean cut through face stock, liner intact, no feathering.
  6. Do not start the main run until QC signs off the First-Off row in Section 3.
C. IN-PROCESS
  1. Every reel change: record start / end meters, operator, and time in Section 4.
  2. Every 30 min or every reel: visual check for streaks, ghosting, misregistration, splice breaks.
  3. Every 60 min: densitometer spot-check on each ink; retighten colour if ΔE > 3.0.
  4. Log any downtime > 5 min in Section 5 with cause code.
  5. Tag and separate any waste reels; do NOT release them into finished stock.
D. POST-RUN
  1. Clean plates with plate-wash; return to plate store with job reference and plate code.
  2. Wash ink decks and anilox rolls; log anilox hours.
  3. Package finished reels with reel ID labels (count, meters, job no., customer).
  4. Retain 3 sample labels in QC retention file (minimum 12 months).
  5. Complete dispatch authorization (Page 3); hand traveller to Production Supervisor.
